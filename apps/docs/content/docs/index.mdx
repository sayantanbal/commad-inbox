# Command Inbox Documentation

Welcome to the official documentation for **Command Inbox**, a keyboard-first Gmail and Google Calendar command center built on the [Corsair SDK](https://corsair.dev).

## Who this docs site is for

| Section | Audience | Goal |
|---------|----------|------|
| [Overview](/docs/overview/introduction) | Judges, evaluators, product reviewers | Understand the product thesis, architecture, and differentiators in minutes |
| [User Guide](/docs/user-guide/getting-started) | End users | Learn keyboard workflows, triage lanes, search, and the agent |
| [Developer Guide](/docs/developer-guide/local-development) | Contributors and self-hosters | Run locally, configure webhooks, deploy to production |
| [Reference](/docs/reference/environment) | Integrators and maintainers | Look up every env var, API route, schema table, and shortcut |

## Quick links

- **Live app:** [command-inbox.sayantanbal.in](https://command-inbox.sayantanbal.in)
- **Hero workflow:** Open a scheduling email → press `M` → pick a slot → invite + confirmation draft
- **Agent example:** *"Send a calendar invite to friend@corsair.dev at 9 AM next Thursday and draft a confirmation reply"*

Start with [Introduction](/docs/overview/introduction) for the product story, or [Getting started](/docs/user-guide/getting-started) if you already have access to the app.
